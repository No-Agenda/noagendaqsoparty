The Rules

  1. Abide by all Amateur Radio regulations in your country.
  2. No Agenda QSO Party Organizers are not responsible for any direct or indirect damage/trouble associated with this QSO Party.
  3. When using repeaters and reflectors, be considerate of other users who are not taking part in the QSO Party.

Making A Call

<Call>
"CQ No Agenda QSO Party 2015"
<Contest Number>
The other station's RS/RST
The year you started listening to No Agenda
Your contributor status (if applicable):
"P" - Producer
"K" - Knight/Dame
"B" - Baron/Baronette
"D" - Duke
"C" - Creator

Points

You earn one point for each QSO made.

You can earn bonus points for the following:

  • 1 point for each unique country contacted.
  • 1 point for each unique U.S. State contacted.
  • 1 point for each Knight/Dame contacted.
  • 2 points for each Baron/Baronette contacted.
  • 3 points for each Duke contacted.
  • 10 points for each Creator contacted.

Submitting Call Logs

We have provided a log book where you can either manually enter your QSOs during the party or upload files using the Cabrillo V3.0 format. You will need to register with the site in order to access the log book. Once registered, you will be able to add as many logs as you wish.

If you plan to upload a file of your QSOs, please include the following information. For each QSO record include only the following fields; frequency, mode, date, time, call sent, rst sent, exchange sent, call rcvd, rst rcvd, exhcnage rcvd and transmitter number. We will use the callsign lookup at QRZ.com to determine each operator's country and state/province to help in reducing the amount of information needed to send in the QSO.

If you have issues using the log book, please email the organizer KF5SVR with the subject #NOAGENDAQSOPARTY CONTEST LOG.
